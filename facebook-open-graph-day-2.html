Facebook Open Graph – Day 2

어제에 이어 페이스북 오픈 그래프 작업을 진행한다.

오픈그래프 액션에 대해서 오픈그래프 오브젝트에 대응하는 트래블로그의 페이지의 Self-Hosted Objects 설정값을 App 의 그것과 일치하게끔 서버 개발자에게 요청 한다. 해당되는 오브젝트 페이지로 로그, 도시, 국가가 있으며, 각각의 오브젝트 설정값은 아래와 같다.

1. Log

  • title - [full name of user]'s log at [place name]  in [city name, country name] // 플레이스 정보가 없을땐 at을 포함해서 생략
  • url - 로그의 웹페이지 url
  • image - 로그의 이미지
  • description - [memo] [tags with '#' and comma(,)]
  • travelogme:country - country name
  • travelogme:city - city name
  • travelogme:location - latitude, longitude
  • place:location - latitude, longitude
  • fb:app_id - app id

2. City

  • title - [city name](, [state name]), [country name]
  • url - City 웹페이지 url
  • image - city의 대표 이미지
  • travelogme:country - country name
  • travelogme:city - city name
  • fb:app_id - app id
