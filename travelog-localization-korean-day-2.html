Travelog Localization (Korean) - Day 2

트래블로그 한국 지역화 작업 중이며, 금일 텍스트에 대한 번역 자료를 모두 입력 했다.

Issue

로그의 시간이 맞지 않는 다는 리포팅이 있었다. 원인은 로그를 작성하는 시점에서 현재의 위치가 아닌 타임존이 다른 다른 국가의 위치를 입력할 경우, 사용자는 작성하는 시점의 국가가 아닌 실제 여행던 국가의 사간을 입력하게 되는데, 이때 작성 시점의 국가와 여행한 국가의 타임존이 달라져서 사용자가 생각한 시간과 다른 시간이 표시되게 되는 것이다.

이 문제에 대해서 로그의 GPS 정보가 바뀔 때 마다 구글의 타임존 API 를 이용해 좌표를 보내서 타임존을 받아와 적용하는 것으로 해결 했다.
